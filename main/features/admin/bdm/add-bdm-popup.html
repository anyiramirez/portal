Warning:

Note that updating the model modifies existing database tables definitively.
Bonita uses Hibernate for data persistence, therefore some changes are handled well, like adding new objects or attributes, but some others we cannot guarantee so far, like changing the type of an attribute.
In such cases, you will have to script the changes, after backuping the database, and through careful validation in a staging environment.
